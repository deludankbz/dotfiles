### Tricks & Stuff
#### Paste clipboard contents to a file in terminal
1. Copy your data.
2. Run `cat > /your/file/path` in the terminal window
3. Paste the contents to the terminal window `Ctrl + Shift + V`
4. Press press `Ctrl + D`
